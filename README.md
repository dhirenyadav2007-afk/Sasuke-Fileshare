## ◈ Overview

**File Store Pro** is a production-grade Telegram file-storage bot built with **Pyrofork** and **MongoDB**. It stores files in private DB channels and delivers them through encoded, expiring links — complete with force-subscription gates, a premium credit system, quality-grouped batch links, and a full channel post manager.

> Built and maintained by [@ITS_shun_x](https://t.me/ITS_shun_x) · Powered by [@BotifyX_Pro_Botz](https://t.me/BotifyX_Pro_Botz)

## ⌗ Command Reference

### ▸ User Commands

| Command | Description |
|---|---|
| `/start` | Launch the bot · receive files via deep links |
| `/profile` | View your credit balance and plan |
| `/credits` | Check remaining credits |
| `/buy` | View premium plans |
| `/request` | Request join approval for FSub channels |

### ▸ Admin Commands — File Management

| Command | Description |
|---|---|
| `/genlink <msg_id>` | Generate a single encoded file link |
| `/batch <first> <last>` | Batch-link a range of DB-channel messages |
| `/fbatch` | Quality-grouped batch — Season × Quality encoded links |
| `/setfbatch` | Configure fbatch assets (custom pic · start sticker · end sticker) |

### ▸ Admin Commands — Post Manager

| Command | Description |
|---|---|
| `/post` | Open the Post Manager panel |
| `/addchnl -100xxx` | Add a post channel (rights verified instantly) |
| `/delchnl -100xxx` | Remove a post channel |
| `/send` | Publish or apply the active post session |
| `/abort` | Cancel any active multi-step session |

### ▸ Admin Commands — Bot Management

| Command | Description |
|---|---|
| `/stats` | Bot statistics |
| `/users` | List all users |
| `/broadcast` | Broadcast message to all users |
| `/pbroadcast` | Broadcast to premium users only |
| `/ban <id>` | Ban a user |
| `/unban <id>` | Unban a user |
| `/addpremium <id>` | Grant premium access |
| `/delpremium <id>` | Revoke premium access |
| `/premiumusers` | List all premium users |
| `/add_credit <id> <n>` | Add credits manually |
| `/shortner` | Toggle shortener on/off |
| `/db` · `/adddb` · `/removedb` | Manage DB channels |
| `/cplan` | View or edit premium credit plans |
| `/bought <id>` | Mark manual payment as received |

## ◈ Architecture

```
File Store Pro
├── bot.py                    ← Bot client (global send_message override for no link previews)
├── config.py                 ← All environment and configuration variables
├── main.py                   ← Entry point
│
├── helper/
│   ├── database.py           ← MongoDB — users, channels, batches, fbatch groups, post channels
│   ├── fbatch_helper.py      ← Filename parser · quality extractor · season grouper · link formatter
│   ├── helper_func.py        ← encode/decode · force_sub decorator · auto-delete · credits
│   └── post_state.py         ← Shared session state (sessions dict + active set) — no circular imports
│
└── plugins/
    ├── start.py              ← /start · deep-link file delivery · fbatch link delivery
    ├── channel_post.py       ← DB-channel forwarder · EXCLUDED_COMMANDS master list
    ├── fbatch.py             ← /fbatch · /setfbatch
    ├── link_generator.py     ← /genlink · /batch
    ├── post.py               ← /post · /send · /abort · /addchnl · /delchnl
    ├── force_sub.py          ← FSub decorator (multi-channel + join request)
    ├── admin.py              ← Admin utilities
    └── ...
```

## ✦ FBatch — Quality Link Flow

```
Admin: /fbatch
       │
       ├─ Forward first DB-channel message (or send link)
       ├─ Forward last DB-channel message
       │
       └─ Bot scans the full range:
               ├─ Groups files by Season then Quality
               │     Quality order: 360p < 480p < 720p < 1080p < 4K < HD < HDRip < BluRay
               ├─ Saves each group to MongoDB  →  fbatch_XXXXXXXXXXXXXXXX
               ├─ Encodes each batch ID (base64, identical to /batch and /genlink)
               └─ Sends admin a copyable code block:

                   SEASON: S01 :~
                   ✧480p✧ - t.me/bot?start=Zmj... | ✧720p✧ - t.me/bot?start=Zmj...
                    ✧1080p✧ - t.me/bot?start=Zmj...

                   SEASON: S02 :~
                   ✧480p✧ - t.me/bot?start=Zmj... | ✧720p✧ - t.me/bot?start=Zmj...

User clicks a quality link:
       ├─ Ban check → Premium check → Shortener/Credit check
       ├─ Info card  (custom_pic + Anime · Season · Audio · Quality · Episodes)
       ├─ Start sticker
       ├─ All files for that quality group
       ├─ End sticker
       └─ Auto-delete countdown begins
```

## ✎ Post Manager — Full Flow

```
/post  →  ▸ ᴘᴏsᴛ ᴍᴀɴᴀɢᴇʀ
  │
  ├─ » ᴄʀᴇᴀᴛᴇ ᴘᴏsᴛ
  │       ├─ Paginated 3×2 channel picker (prev / next / back)
  │       ├─ Send content  →  text · photo · photo+caption · file
  │       ├─ Add buttons?
  │       │     ├─ ✎ Manual format
  │       │     │     Season 01 - SEASON 01        ← popup alert button
  │       │     │     480p - https://... | 720p - https://...
  │       │     │     Season 02 - SEASON 02
  │       │     │     480p - https://... | 720p - https://...
  │       │     ├─ ▦ Paste /fbatch output  (auto-parsed)
  │       │     └─ Skip buttons
  │       ├─ Real preview with live inline buttons sent to admin DM
  │       └─ /send  →  "✓ Post sent to ChannelName · View Post"
  │
  ├─ ✎ ᴇᴅɪᴛ ᴘᴏsᴛ
  │       ├─ Forward channel post or paste its t.me/c/xxx/xxx link
  │       ├─ Choose:  ▣ Media/Text   OR   ◉ Buttons
  │       └─ /send  →  "✓ Post edited in ChannelName · View Post"
  │
  ├─ ◈ ᴄʜᴀɴɴᴇʟ sᴛᴀᴛs
  │       └─ Paginated grid of all post channels
  │               └─ Click any  →  Name · Channel ID · Added On
  │
  └─ /abort  →  cancels any session at any point
```

> **Session Guard** — while any session is active, the DB-channel auto-forward is suspended for that admin. No accidental file saves during post building or fbatch runs.

## ◷ Setup & Deployment

### Prerequisites

```
Python 3.10+   ·   MongoDB URI   ·   Telegram API ID + API HASH   ·   Bot Token
```

### ⬡ Local

```bash
git clone https://github.com/yourname/File-Store-Pro
cd File-Store-Pro
pip install -r requirements.txt
# edit config.py — fill TOKEN, API_ID, API_HASH, DB_URI, DB_CHANNEL, OWNER_ID
python main.py
```

### ⬡ Docker

```bash
docker-compose up -d
```

### ⬡ Heroku / Render

Use the included `Procfile` / `render.yaml`. Set all config values as environment variables in the dashboard.

## ◈ Configuration Reference

| Variable | Description | Required |
|---|---|---|
| `TOKEN` | Bot token from [@BotFather](https://t.me/BotFather) | ✓ |
| `API_ID` | From [my.telegram.org](https://my.telegram.org) | ✓ |
| `API_HASH` | From [my.telegram.org](https://my.telegram.org) | ✓ |
| `DB_URI` | MongoDB connection string | ✓ |
| `DB_NAME` | MongoDB database name | ✓ |
| `DB_CHANNEL` | Primary DB channel ID e.g. `-100xxxxxxxxx` | ✓ |
| `OWNER_ID` | Your Telegram user ID | ✓ |
| `ADMINS` | List of admin user IDs `[123, 456]` | ✓ |
| `FSUBS` | FSub config `[[channel_id, request_mode, timer_min]]` | ✓ |
| `AUTO_DEL` | File auto-delete delay in seconds · `0` = disabled | — |
| `PROTECT` | Content protection — blocks forward and screenshot | — |
| `DISABLE_BTN` | Strip inline buttons from forwarded files | — |
| `SHORT_URL` | Shortener domain e.g. `shrinkme.io` | — |
| `SHORT_API` | Shortener API key | — |
| `WORKERS` | Pyrogram worker thread count | — |
